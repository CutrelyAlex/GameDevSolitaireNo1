【活动规则】
活动分为以下几个阶段：
1、报名阶段（2.9-2.13）
2、准备阶段（2.14-2.15）
3、开发阶段（2.16-2.22）
4、内部体验阶段（2.23-2.28）
5、对外宣传阶段（3.1-3.7）
6、评奖阶段（3.8）

1、报名阶段
主要是收集开发者信息，看看有多少人过年比较无聊愿意来玩儿，鉴于开发周期为一周，所以我们最多不要超过7名开发者。3~7人是比较合适的。

2、准备阶段
作为第一届的主办方，凯恩会在拿到开发者信息之后进行筛选并确定最终的参赛人员名单和开发顺序。之后凯恩会给第一位开发者提供一个关键词，由开发者自行决定是否使用该关键词进行项目立项。
*鉴于这是一次测试活动，所以我们限定玩法为【文字冒险】，引擎根据报名的参赛人员取最大公约数。

3、开发阶段
根据人数计算每个开发者可以进行的开发时长，到时间必须进行提交，之后下一个开发者拉取工程并继续开发。
每个开发者都能看到截至目前的所有开发进度，但是禁止跟前任开发者进行任何形式的场外沟通，也不能给后续的开发者留下任何形式的场内提示，也就是说，参与者们完全不知道这个项目究竟是关于什么主题的。
到2月22号晚23:59分，最后一名开发者需要完成打包并提交所有的开发内容，开发阶段到此为止，之后进入内部体验环节。

4、内部体验阶段
所有的群友都可以来体验这个游戏并发表自己的感想，每一个群友都有一票来选出自己最喜欢的部分（但是群友不知道这一部分是由哪位开发者开发的），凯恩会记录这些评价信息和投票数据用于之后的评奖。

5、对外宣传阶段
之后凯恩会录制并剪辑一个活动说明视频和游戏宣传视频上传B站，然后把包体放到独游指针的官方群里（因为凯恩的群是开发者群，不方便放很多的玩家朋友们进来），所有感兴趣的玩家都可以进群下载包体游玩并在调查问卷中留下自己宝贵的一票。

6、评奖阶段
3月8号，凯恩会整理好所有的投票数据并公布最终的开发人员、开发顺序和得票情况。
之后进行线上颁奖活动，人气最高的开发者会获得Kenner老师提供的100块红包。（后续是否有更多的奖品，我们可以一起期待一下）

其他注意事项：
·每个开发者都通过该仓库提交所有的资产，该仓库永久public，任何人都可以拉取和查看（但合入受限）
·本次活动限定玩法为文字冒险游戏，游戏整体的叙事要保证是线性结构或者珍珠链结构，每个开发者要在自己负责的流程内收束所有的选项并保留唯一后置选择给下一名开发者，这样可以有效避免因为过于丰富的选择导致后接手的开发者工作量指数级上升
·开发者可以新增美术资产或者利用现有资产，不可以删除其他人的美术资产
